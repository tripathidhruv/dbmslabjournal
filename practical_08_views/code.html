Practical 08:
The Virtual Layer (Views)

Exploring the abstraction of data through SQL Views—enhancing security, simplifying complexity, and managing virtual datasets.

Objective

Understand how to implement and manage virtual tables in a relational database system using the CREATE VIEW statement.

  • check_circle Data Abstraction
  • check_circle Logical Independence
  • check_circle Security Through Redaction
Technical Brief

Virtualization of Storage

01

Creating Basic Views

A view is a virtual table whose contents are defined by a query. It does not exist as a physical set of data values.

SQL Syntax
CREATE VIEW employee_summary AS
SELECT e.first_name, e.last_name, d.dept_name
FROM employees e
JOIN departments d ON e.dept_id = d.dept_id
WHERE e.salary > 50000;

Updatable Views

Views can be used to update the underlying tables if they satisfy certain conditions like containing a primary key and having no aggregate functions.

Core Restrictions

You cannot use ORDER BY (unless TOP is specified) or temporary tables within a view definition in most standard SQL dialects.

02

Views from Multiple Tables

Complex Join View
CREATE OR REPLACE VIEW lab_inventory AS
SELECT s.supplier_name, i.item_name, c.category_name
FROM items i
INNER JOIN suppliers s ON i.supp_id = s.supp_id
LEFT JOIN categories c ON i.cat_id = c.cat_id;

Viva Q&A

What is the difference between a Table and a View? expand_more
A Table stores physical data on the disk, whereas a View is a logical structure that stores only the query definition. A View behaves like a table but computes results on the fly.
Can we use DML operations on all Views? expand_more
No. Only "Simple Views" (those created from a single table without groupings or functions) are generally updatable. Complex views involving joins or aggregates are usually read-only.
What is 'WITH CHECK OPTION'? expand_more
It ensures that all UPDATE and INSERT statements satisfy the condition(s) in the view definition. It prevents users from updating rows such that they disappear from the view.
