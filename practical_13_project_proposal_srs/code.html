Project Proposal (SRS)

Formalizing software requirements and technical documentation for the Online Library Management System.

target System Goals

  • 01 Automate the legacy manual book tracking system to reduce human error and data redundancy.
  • 02 Implement a real-time availability checker for over 50,000+ digital and physical assets.
  • 03 Streamline the fine calculation and digital payment gateway integration for overdue returns.
